This form allows for printing of text only Magic: The Gathering proxy paper inserts. This is useful for putting into a sleeve with a regular Magic card for playtesting and so on. Card names can be prefixes with a number to add that many to the printed sheet.

Note that that page is in no way affiliated with Wizards of the Coast, Hasbro, or any of its affiliates. This functionality is provided as in and follows the "approve or improve" principle. The code is available on GitHub.
